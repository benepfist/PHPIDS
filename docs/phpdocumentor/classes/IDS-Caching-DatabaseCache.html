DatabaseCache.php : 72

Database caching wrapper

This class inhabits functionality to get and set cache via a database.

Needed SQL:

#create the database

CREATE DATABASE IF NOT EXISTS phpids DEFAULT CHARACTER SET utf8 COLLATE utf8_general_ci; DROP TABLE IF EXISTS cache;

#now select the created datbase and create the table

CREATE TABLE cache ( type VARCHAR( 32 ) NOT null , data TEXT NOT null , created DATETIME NOT null , modified DATETIME NOT null ) ENGINE = MYISAM ;
